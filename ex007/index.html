<u>...</u> - É uma tag para colocar textos de forma sublinhada, é importante manter uma atenção especial com essa tag, poís ela corre risco de ficar obsoleta com as formas de estilização em CSS

<address>...</address> - É uma tag para incluir endereços no HTML, referenciando possívelmente até um link

<style> - É uma tag de referência rapida ao CSS, para comandos em CSS mais curtos, como mudar a cor de fundo da pagína por exemplo, usada dentro do campo <head>
