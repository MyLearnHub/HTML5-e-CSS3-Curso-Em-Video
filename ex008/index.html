<b>...</b> - É a tag em inglês para escrever textos em negrito, porém é conhecida por ser uma tag não semântica, logo tem possíbilidade de cair em desuso brevemente

<strong>...</strong> - É a tag para atribuir um significado de destaque dentro de um texto, ou seja, a forma correta de se colocar algo em negrito

<i>...</i> - É a tag em inglês para escrever textos em itálico, porém é conhecida por ser uma tag não semântica, logo tem possíbilidade de cair em desuso brevemente

<em>...</em> - É a tag para atribuir um significado de destaque dentro de um texto, ou seja, a forma correta de se colocar algo em itálico

<mark>...</mark> - É a tag para atribuir uma marcação sobreposta ao texto, como um marca texto, e para alterar sua cor se usa uma tag 'style="background-color: ...;"' para mudar a cor do marcador.

<small>...</small> - É a tag para atribuir o tamanho de uma letra como pequena

<big>...</big> - É a tag para atribuir o tamanho de uma letra como grande, assim como outras tags essa já está obsoleta

<del>...</del> - É a tag para destacar um texto, de forma que ele fique riscado, ou indicando que foi deletado

<ins>...</ins> - É a tag para destacar um texto, de forma que ele fique sublinhado, ou indicando que foi inserido

<u>...</u> - É a tag para sublinhar um texto, também pode se tornar obsoleta no futuro

<sup>...</sup> - É a tag para deixar um texto sobrescrito

<sub>...</sub> - É a tag para deixar um texto subescrito
