<code>...</code> - É uma tag para destacar um texto como um código

<pre>...</pre> - É uma tag normalmente utilizada com a tag <code>, para auxiliar na identação, considerando os espaços

<q>...</q> - É uma tag que identifica um texto como citação simples, inserindo aspas

<blockquote>...</blockquote> - É uma tag que identifica um texto como citação, e ainda podendo ser usada como parte de referência a um link com a fonte original, utilizando o comando 'cite=""'

<abbr>...</abbr> - É uma tag que identifica um texto como abreviação simples, sublinhando e mostrando o contéudo completo definido no comando 'title=""'

<bdo>...</bdo> - É uma tag que orienta a direção de um texto, da esquerda para direita ou vice versa, definida pelo comando 'dir=""'
