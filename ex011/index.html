Mídias



<picture>...</picture> - É a tag para mídias, utilizada aqui para criar uma imagem dinâmica, inserindo a tag <source>, se especifica a largura maxíma de cima para baixo para alterar as imagens, no atributo 'media=""', especificando a imagem no atributo 'srcset=""' e colocando o tipo da imagem no atributo 'type="image/"'

<audio src="">...</audio> - É a tag para aúdio no HTML, ela funciona semelhante a tag <img>, mas podendo ainda ter dentro de si a tag <source>, alguns dos seus atributos são 'loop', para manter ela em looping, 'autoplay', para fazer tocar assim que inciar a página, 'controls', para ter acesso aos controles do aúdio e sempre sendo especificado seu tipo no atributo 'type=""'
