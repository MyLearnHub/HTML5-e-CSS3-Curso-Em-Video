Estilos CSS Inline

Usar a tag <style> dentro de uma tag HTML faz com que ela funcione quase que como um atributo, podendo alterar tamanho, cor, fonte, etc. Sendo utíl para estilizações de pagínas menores, mas pouco eficiente de modo geral, já que polui a pagína com códigos CSS, promovendo a desorganização

background-color:...; - É a propriedade CSS que define a cor de fundo de algo

color:...; - É a propriedade CSS que define a cor de algo

font-family:...; - É a propriedade CSS que define a fonte de um texto

font-size:...; - É a propriedade CSS que define o tamanho da fonte de um texto

text-align:...; - É a propriedade CSS que define o alinhamento de um texto
